#          Key-Management-System  

## 项目介绍：

任务：设计一个密钥管理系统，可以完成密钥生成和管理。

**基本要求：** 

（1）可以生成随机数，必要时生成大素数； 

（2）检测密钥和密码的安全性能； 

（3）密钥的加密保存和管理；

（4）生成公钥对；

（5）可以对密钥进行签名。

## 项目内容：

分别选择了非对称加密算法 RSA 和对称加密算法 AES 进行密钥管理。

可以选择自定义设置密钥，也可以选择系统生成。

设计检测算法，对 RSA 密钥强度和 AES 密钥强度进行检验。

系统内部对密钥使用 RSA 进行数字签名，使用 SHA256 进行加密储存。

密钥以 json 格式储存。
